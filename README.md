# human-api-app
Sample flask based APIs for testing and automation practice


# Get the swagger UI
- http://<host_name>:<port_number>/api/ui/#/People/people.read_all

# Setup

- create virtual env
- install all requirements from requirement.txt
- run "python server.py" command
